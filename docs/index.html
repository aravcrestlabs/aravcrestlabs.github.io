GemCrest Documentation

Welcome to the official documentation for GemCrest, the elegant jewelry billing software by AravCrestLabs.

Quick Overview

GemCrest is a professional-grade billing and inventory management solution designed exclusively for jewelry businesses in India. It combines powerful features with an intuitive interface to help you manage your shop efficiently.

Key Features

  • GST-Compliant Invoicing: Generate professional invoices with all tax calculations handled automatically
  • Inventory Management: Track every item with weight, purity, HUID, and stock levels
  • Customer CRM: Maintain customer profiles with purchase history
  • Metal Rates: Update daily gold/silver rates for accurate pricing
  • 100% Offline: All data stays on your device, no internet required
